## 구구단 작성
코드 구현를 구현 함.
 - <p id = 'result'>구구단</p>
 - var p = document.**getElementById**('result');
 - getElementById 를 이용해서 아이디를 찾고 정의함
 - p.**innerHTML** = str;
 - innerHTML을 사용해서 id가 'result' 를 str로 변경
